## Common Mixins

Almost all tables in SynqX inherit from these base mixins to ensure consistent auditing.

### `TimestampMixin`
Automatically tracks creation and update times.
-   `created_at`: `DateTime` (Immutable)
-   `updated_at`: `DateTime` (Auto-update)

### `UserTrackingMixin`
Tracks the actor responsible for changes.
-   `created_by`: `String`
-   `updated_by`: `String`

### `TenantMixin`
Enforces security. **Every critical query must filter by `user_id`.**
-   `user_id`: `String(64)` (Indexed)

### `SoftDeleteMixin`
Allows for safe deletion and recovery.
-   `deleted_at`: `DateTime` (Nullable)
-   `deleted_by`: `String`
-   `is_deleted`: `property` (Checks if `deleted_at` is not null)
